# Quiz App

A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript. The application presents multiple-choice questions, tracks the user's score, and displays the final result after the quiz is completed.

## Features

* Interactive multiple-choice quiz
* Real-time score tracking
* Clean and responsive user interface
* Instant result display after quiz completion
* Easy to customize with additional questions

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
quiz-app/
│── style.css
│── script.js
├── index.html
├── README.md
├── LICENSE
```

## Installation

1. Clone this repository.

```bash
git clone https://github.com/your-username/quiz-app.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional installation or dependencies are required.
